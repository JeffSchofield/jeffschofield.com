AI Holiday Card Generator

I was pulled in to work on this project in my second week at DesignStripe. It is an experimental AI-powered holiday card generator that creates both an image and some text relevant to the holiday and subjects you choose. To generate the images, it uses a Stable Diffusion model that was fine-tuned on holiday cards by our in-house AI expert. The greetings text is generated by GPT-3.

I was tasked with making sure the design was responsive for all kinds of device sizes, as well as creating an algorithm to make sure the card text would resize to never overflow the card’s boundaries while maintaining the same aesthetic quality of design.
