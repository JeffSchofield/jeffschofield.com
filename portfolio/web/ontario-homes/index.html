Ontario Homes

Ontario Homes is a new real estate photography service in the Cobourg area owned and operated by my brother. We worked together on brand aesthetics and came up with a single page website that effectively communicates services, prices, and examples to the desired customer base.

As a photograph-heavy website, I put a lot of effort in to reducing the page size by using modern browser features to produce many variations of each image in different sizes and formats for the browser to select from. The hero background video is also compressed and formatted for both desktop and mobile experiences to avoid eating away bandwidth. I was careful to select a focal point for the video that would work well on both desktop and mobile aspect ratios when cropping from the source video.

The project is built with Vue 3 and Tailwind CSS. I used Vite to pre-render the project into static HTML files which are then hosted using GitHub Pages.
