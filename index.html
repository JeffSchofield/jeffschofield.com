Hello! My name is Jeff Schofield. I am a soft­ware developer from Peter­borough, Ontario who special­izes in application develop­ment for the web. I have 10 years of pro­fessional exp­erience designing and con­structing websites, web applications, as well as the server software and infra­structure that powers it all.

Front, back, and
everything in between.

I have been working on websites and applications of varying size and complexity since my teenage years. With a tool kit put together by experience and familiarity of common web software patterns, I am comfortable designing simple static websites, developing prototypes for more dynamic web applications, and working on existing codebases to add new features or update existing ones.

Beautiful and functional interfaces.

I use modern reactive frameworks and systems of design to create eye-popping marketing pages and application interfaces with low cognitive overhead alike. I also love working with WebGL to create next-level visualizations that leave a lasting impression.
